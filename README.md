一个简单的脚本，用于替换 Windows 的字体。

字体来源：[更纱黑体](https://github.com/be5invis/Sarasa-Gothic) - [大佬修改版](https://bbs.pcbeta.com/viewthread-1960120-1-4.html)

工具来源：[字体替换工具](https://www.fishlee.net/soft/sysfontreplacer/)



使用前，在 `replace-font-en` 和 `replace-font-zh` 下放置替换的英文和中文字体。

然后运行 `backup_fonts.py`，备份与替换字体同名的系统字体。

然后使用 `字体替换工具`。

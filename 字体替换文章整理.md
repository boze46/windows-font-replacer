#### Windows 字体链接修正

Windows 10/11 的中文字体默认缺少 Semilight、Semibold、Heavy，但 Segoe UI 提供 6 个字重，可能产生链接错误。注册表位置：

`HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\FontLink\SystemLink`

`text
Segoe UI Semibold：
MSYHSB.TTC,Microsoft YaHei UI Semibold,128,96
MSYHSB.TTC,Microsoft YaHei UI Semibold

Segoe UI Semilight：
MSYHSL.TTC,Microsoft YaHei UI Semilight,128,96
MSYHSL.TTC,Microsoft YaHei UI Semilight

新增 Segoe UI Black：
MSYHHV.TTC,Microsoft YaHei UI Heavy,128,96
MSYHHV.TTC,Microsoft YaHei UI Heavy
`

Segoe UI Variable 的 Display、Small、Text 等 Semibold/Semilight 变体可参照处理，注销后生效。

## 五、核心结论

1. Hinting 改善低分辨率屏幕下的字形清晰度。
2. 字体卡顿不一定由 hinting 引起；微软雅黑 11.3 的主要问题是异常庞大的 GSUB 表。
3. 中英文字重应成套匹配，否则可能出现视觉重量不一致。
4. Windows FontLink 会影响 Segoe UI 与微软雅黑之间的字重匹配。
5. 替换宋体需权衡屏幕阅读、办公排版和打印效果。
6. 字体名称、嵌入权限、CVT、GSUB、竖排替换都会影响兼容性。

## 六、安全提示

系统字体替换和注册表修改属于高风险操作，可能造成乱码、应用异常、无法正常登录或系统更新问题。

1. 创建系统还原点，并备份 `C:\Windows\Fonts` 中的原字体。
2. 导出 `FontLink\SystemLink` 注册表键。
3. 不要混装不同日期、不同系列的同名字体。
4. 替换后注销或重启，并检查浏览器、Office/WPS、Adobe 和常用软件。
5. 出现乱码时优先恢复原字体。
